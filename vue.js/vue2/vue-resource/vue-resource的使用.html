基本语法

引入vue-resource后，可以基于全局的Vue对象使用http，也可以基于某个Vue实例使用http

基于全局Vue对象使用http

Vue.http.get('/someurl',[options]).then(successCallback,errorCallback)

Vue.http.post('/someurl',[body],[options]).then(successCallback,errorCallback)

在vue实例里使用$http

this.$http.get('/someurl',[options]).then(successCallback,errorCallback)

this.$http.post('/someurl',[options]).then(successCallback,errorCallback)

在发送请求后，使用then方法来处理响应结果，第一个是响应成功时的回调函数，第二个参数是响应失败时的回调函数。

回调函数也有两种写法，一种是传统的函数写法，第二种是es6的写法(箭头函数)

this.$http.get('someurl',[body],[options]).then(function(response){ },function(response){ })
this.$http.get('someurl',[options]).then((response)=>{ },(response)=>{ })
